Sales report by region

This module aims to provide you best advantage of sale report by giving you detailed sales report by state, from which you can know which geographic area is the best to concentrate on.

Configuration

you don't need any configuration, only you need to set state(geographic area) on each partner

Screenshots

Go to Sales/Orders/ Customers menu , and on Address section set State field

Go to Sales/Orders/ Orders menu , and go to pivot view, by default Sales are detailed by Customer states as columns on pivot

Go to Sales/Reporting / Sales menu also notice sales are defined by states by default

Also you can filters your sales report by using state
